# Release Checklist

- Confirm `README.md` and `README.zh-CN.md` are both updated.
- Confirm install, uninstall, rollback, and export scripts still work.
- Confirm no personal secrets or machine-specific paths are committed.
- Confirm attribution in `NOTICE.md` is still accurate.
- Tag the first public release as `v0.1.0`.
